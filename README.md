# ระบบจัดการแผนการสอน - โรงเรียนลำไพลศานติวิทย์

ระบบเว็บแอปพลิเคชันสำหรับจัดการแผนการสอนและบันทึกหลังสอน ออกแบบตามหลักการ Active Learning และ Reflective Practice

## 🎯 คุณสมบัติหลัก

### สำหรับครู
- ✅ สร้างและแก้ไขแผนการสอน (ส่งทุก 2 สัปดาห์)
- ✅ เขียนบันทึกหลังสอน (Reflection)
- ✅ ดูสถานะการอนุมัติ
- ✅ พิมพ์แผนและบันทึกออกมาเป็นเอกสาร
- ✅ ดูสถิติส่วนตัว

### สำหรับฝ่ายวิชาการ
- ✅ ตรวจสอบแผนการสอนจากครู
- ✅ ตรวจสอบบันทึกหลังสอน
- ✅ ให้ความคิดเห็น/ส่งแก้ไข
- ✅ ส่งต่อให้ ผอ. อนุมัติ

### สำหรับผู้อำนวยการ
- ✅ อนุมัติ/ส่งแก้ไขแผนการสอน
- ✅ อนุมัติ/ส่งแก้ไขบันทึกหลังสอน
- ✅ ดูรายงานสถิติภาพรวมโรงเรียน
- ✅ ดูสถิติรายบุคคลของครูทุกคน
- ✅ **จัดการผู้ใช้งาน** (เพิ่ม/ลด/แก้ไข)
  - กำหนดบทบาท (ครู/วิชาการ/ผอ.)
  - กำหนดตำแหน่ง
  - กำหนดสังกัด/กลุ่มสาระ
  - กำหนดระดับชั้นที่สอน
  - เปิด/ปิดใช้งานบัญชี

## 🚀 การติดตั้งบน GitHub Pages

1. **Fork หรือ Clone** repository นี้
2. ไปที่ **Settings** > **Pages**
3. เลือก **Source**: Deploy from a branch
4. เลือก **Branch**: main (หรือ master)
5. กด **Save**
6. รอสักครู่ เว็บไซต์จะพร้อมใช้งานที่ `https://[username].github.io/[repo-name]`

## 💾 การจัดเก็บข้อมูล

ระบบใช้ **localStorage** ในการเก็บข้อมูล ซึ่งหมายความว่า:
- ข้อมูลจะถูกเก็บไว้ในเบราว์เซอร์ของผู้ใช้แต่ละคน
- ข้อมูลจะคงอยู่แม้ปิดเบราว์เซอร์
- ข้อมูลจะหายไปถ้าล้าง browser data
- เหมาะสำหรับการทดลองใช้งานหรือใช้งานขนาดเล็ก

⚠️ **หมายเหตุ**: สำหรับการใช้งานจริงในโรงเรียน แนะนำให้เชื่อมต่อกับฐานข้อมูลจริง เช่น Firebase, Supabase หรือ Backend Server

## 📐 หลักการออกแบบ

ระบบนี้ออกแบบตามทฤษฎี:

1. **Reflective Practice (Donald Schön)**
   - Reflection-for-action: คิดก่อนสอน
   - Reflection-on-action: สะท้อนคิดหลังสอน

2. **Backward Design (Wiggins & McTighe)**
   - เริ่มจากสมรรถนะปลายทาง
   - ออกแบบการประเมิน
   - ออกแบบกิจกรรม

3. **Active Learning**
   - นักเรียนมีส่วนร่วมทางปัญญา
   - เน้นการลงมือทำ

4. **Competency-Based Education**
   - เน้นสิ่งที่นักเรียน "ทำได้"
